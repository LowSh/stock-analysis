# Stock Analysis

股票分析 skill，用于追踪财经事件、分析热门板块、推荐股票并记录到日志文件。

## 工作流程

1. **获取财经日历** - 从财联社 (cls.cn)、同花顺 (10jqka.com.cn) 抓取近期重要事件和新闻
2. **获取热门新闻链接** - 抓取点击量高的相关新闻，附在事件后面
3. **分析事件驱动** - 根据事件识别可能受益的板块
4. **推荐股票** - 每个板块推荐 3 只相关股票（含代码和理由）
5. **技术分析** - 运行脚本获取买入卖出建议
6. **更新日志文件** - 追加到用户指定的日志文件

## 数据来源

| 网站 | 用途 | URL |
|------|------|-----|
| 财联社 | 财经日历、重要事件 | https://www.cls.cn |
| 同花顺 | 热门新闻、点击排行 | https://www.10jqka.com.cn |
| 新浪财经 | 行情数据 | https://finance.sina.com.cn |
| Akshare | 股票技术数据 | Python库 |

## 板块分析（纯结构化框架）

使用 `scripts/analyze_sector.py` 提供板块分析框架。

**✨ 设计理念：**
- **脚本只提供结构化框架**（145行代码）
- **不包含任何预设数据**（零预设板块）
- **所有数据由 AI 大模型动态获取**
- **所有深度分析由 AI 大模型完成**

### 运行方式

```bash
python3 scripts/analyze_sector.py <板块名称>
```

**示例：**
```bash
python3 scripts/analyze_sector.py AI
python3 scripts/analyze_sector.py 黄金
python3 scripts/analyze_sector.py 光伏
```

- 关键词

**2. 龙头股（待AI查询）**
- 提示使用东方财富/同花顺查询

- 新闻分析维度
- 时事结合方法

**4. 输出格式建议**

## 个股分析（单只股票深度分析）

使用 `scripts/analyze_stock.py` 对单只股票进行多维度深度分析。

### 运行方式

```bash
python3 scripts/analyze_stock.py <股票代码> [股票名称]
```

**示例：**
```bash
# 分析广东鸿图
python3 scripts/analyze_stock.py 002101 广东鸿图

# 分析中国石油
python3 scripts/analyze_stock.py 601857 中国石油
```

### 分析维度

| 维度 | 内容 |
|------|------|
| **基本面** | 行业、市值、PE、PB、主营业务 |
| **技术面** | MA5/MA10/MA20、趋势、支撑阻力位 |
| **新闻面** | 近期相关新闻、利好利空分析 |
| **综合评估** | 评分、风险等级、投资周期建议 |
| **操作建议** | 买入/卖出/观望建议 |

**响应流程：**

## 技术分析（买入卖出建议 - 批量分析）

使用 `scripts/stock_analyzer.py` 批量获取持仓股票的技术指标和操作建议。

### 运行方式

```bash
python3 scripts/stock_analyzer.py
```

### 输出格式

```markdown
## 💰 买入卖出建议

| 股票 | 代码 | 现价 | 涨跌% | 买入区间 | 止损价 | 目标价 | 趋势 | 建议 |
|------|------|------|-------|---------|--------|--------|------|------|
| 新洁能 | 605111 | 47.67 | -2.54% | 47.58-47.58 | 45.05 | 52.43 | 多头排列 | 持有/回调加仓 |
```

### 技术指标说明

| 指标 | 说明 |
|------|------|
| MA5 | 5日均线 |
| MA10 | 10日均线 |
| MA20 | 20日均线 |
| 买入区间 | MA5-MA20之间的价格区间 |
| 止损价 | 近期低点下方3% |
| 目标价 | 近期阻力位或+10% |

### 趋势判断

- **多头排列**：MA5 > MA10 > MA20，股价在均线上方 → 持有/加仓
- **空头排列**：MA5 < MA10 < MA20，股价在均线下方 → 观望/减仓
- **震荡**：均线交织，方向不明 → 观望

## 快速开始

### 获取今日财经事件

```
抓取财联社财经日历，获取近期重要事件
```

使用 `web_fetch` 抓取以下网站：
- `https://www.cls.cn` - 财经日历
- `https://www.10jqka.com.cn` - 热门新闻

提取事件、日期和相关新闻链接。

### 获取热门新闻链接

从同花顺首页抓取点击量高的新闻，格式：

```markdown
| 日期 | 事件 | 相关链接 | 板块 |
|------|------|---------|------|
| 3/2 | MWC 2026开幕 | [华为发布新品](https://...) | 通信 |
```

**链接格式要求：**
- 使用 Markdown 格式：`[标题](URL)`
- 优先选择点击量高、时间新的新闻
- 每个事件最多附 2-3 个链接

### 分析热门板块

根据事件类型匹配板块：

| 事件类型 | 相关板块 |
|---------|---------|
| 华为/苹果发布会 | 消费电子、通信 |
| 英伟达/AI大会 | AI、算力、GPU |
| 半导体涨价 | 功率半导体、PCB |
| 两会召开 | 政策概念股 |
| 欧佩克会议 | 石油、油服 |
| 稀土政策 | 稀土、有色 |
| 房地产政策 | 地产、建材 |

### 推荐股票格式

每个板块推荐 3 只股票，**包含技术分析数据**，格式如下：

```markdown
#### X. 🔥 板块名称
**驱动因素：** 具体事件

**推荐股票：**
| 股票 | 代码 | 现价 | 涨跌% | 趋势 | 建议 | 理由 |
|------|------|------|-------|------|------|------|
| xxx | 000000 | 50.00 | +1.5% | 多头排列 | 持有/回调加仓 | 简要说明 |
```

**运行技术分析脚本获取数据：**
```bash
python3 scripts/stock_analyzer.py
```

脚本会自动获取：
- 现价、涨跌幅
- 趋势（多头排列/空头排列/震荡/偏多/偏空）
- 操作建议（持有/加仓/轻仓试探/观望/减仓）
