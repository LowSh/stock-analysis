# Stock WeChat Publisher

生成适合微信公众号的股票分析日报，支持定时自动更新。

## 功能特点

- ✅ **公众号优化格式** - 标题吸睛、分段清晰、emoji丰富
- ✅ **无个人持仓** - 自动去除个人持仓信息，适合公开分享
- ✅ **简洁内容** - 重点突出，适合移动端阅读
- ✅ **热点聚焦** - 突出当日重大事件和热门板块
- ✅ **定时更新** - 支持自动定时更新（可开启/关闭）

## 定时任务管理

### 用户交互方式

**用户说：**
- "开启定时更新" → 运行 `scripts/schedule_manager.sh enable`
- "关闭定时更新" → 运行 `scripts/schedule_manager.sh disable`
- "查看定时状态" → 运行 `scripts/schedule_manager.sh status`
- "立即执行一次" → 运行 `scripts/schedule_manager.sh trigger`
- "设置更新频率为X小时" → 运行 `scripts/schedule_manager.sh set-frequency Xh`

### 使用脚本管理（推荐）

**查看状态：**
```bash
scripts/schedule_manager.sh status
```

**开启定时（每2小时）：**
```bash
scripts/schedule_manager.sh enable
```

**关闭定时：**
```bash
scripts/schedule_manager.sh disable
```

**立即执行一次：**
```bash
scripts/schedule_manager.sh trigger
```

**设置频率：**
```bash
# 每1小时
scripts/schedule_manager.sh set-frequency 1h

# 每4小时
scripts/schedule_manager.sh set-frequency 4h

# 每天9点
scripts/schedule_manager.sh set-frequency "0 9 * * *"

# 交易日（周一到周五，9点和15点）
scripts/schedule_manager.sh set-frequency "0 9,15 * * 1-5"
```

### 直接使用 OpenClaw 命令

### 查看定时状态

**用户说：** "查看定时更新状态"

**操作：**
```bash
openclaw cron list | grep stock-wechat
```

### 开启定时更新

**用户说：**
- "开启定时更新"
- "启动定时任务"
- "每2小时自动更新"

**操作：**
```bash
openclaw cron add --name stock-wechat-every2h --every 2h --message "生成公众号版股票分析" --description "每2小时自动生成公众号版股票分析"
```

**其他频率：**
```bash
# 每天 9:00
--cron "0 9 * * *"

# 每小时
--every 1h

# 每4小时
--every 4h

# 交易日（周一到周五，9/11/13/15点）
--cron "0 9,11,13,15 * * 1-5"
```

### 关闭定时更新

**用户说：**
- "关闭定时更新"
- "停止定时任务"
- "暂停自动更新"

**操作：**
```bash
# 先查询任务ID
openclaw cron list

# 禁用任务
openclaw cron disable <任务ID>

# 或删除任务
openclaw cron delete <任务ID>
```

### 立即执行一次

**用户说：** "立即更新公众号"

**操作：**
```bash
# 先查询任务ID
openclaw cron list

# 立即执行
openclaw cron trigger <任务ID>
```

## 工作流程

1. **获取最新数据**
   - 从财联社、同花顺抓取新闻和事件
   - 运行技术分析脚本（可选）

2. **筛选重点内容**
   - 当日重大事件（地缘政治、政策、行业动态）
   - 热门板块和推荐股票
   - 强势股列表
   - 近期事件预告

3. **生成公众号格式**
   - 吸睛标题（带emoji）
   - 导语摘要
   - 分段清晰（2-3级标题）
   - 简化表格（关键信息）
   - 风险提示

4. **输出文件**
   - 保存到：`~/Desktop/claw实验/股票分析日志_公众号版.md`

## 格式要点

### ✅ 要做的

- **标题吸睛** - 用emoji和热点事件
- **导语简洁** - 20-30字总结核心
- **分段清晰** - 2-3级标题，每段3-5行
- **表格简化** - 只保留关键信息，最多4列
- **emoji丰富** - 🔥 📈 📉 🛢️ 🥇 🔫 📱 📅 💰 ⚠️
- **重点加粗** - 关键信息、涨停股票

### ❌ 不要的

- ❌ **个人持仓** - 所有个人持仓信息
- ❌ **详细技术指标** - MA5/MA10等详细数据
- ❌ **长表格** - 超过4列的表格
- ❌ **过多股票** - 每个板块最多3只
- ❌ **复杂格式** - 公众号不支持的功能

## 与 stock-analysis 的关系

**stock-analysis** 生成完整日志（含个人持仓）
**stock-wechat** 生成公众号版本（无个人持仓）

**推荐工作流：**
1. 先运行 `stock-analysis` 更新完整日志
2. 再运行 `stock-wechat` 生成公众号版本
